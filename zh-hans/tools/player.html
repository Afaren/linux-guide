安装媒体播放器

我们推荐同时安装 VLC media player 和 SMPlayer 两款媒体播放器，这样当遇到无法播放或者播放有问题的视频时能够有多一种选择。

安装 VLC media player

sudo apt install vlc

安装 SMPlayer

sudo add-apt-repository ppa:rvm/smplayer
sudo apt update
sudo apt install smplayer smplayer-themes smplayer-skins
