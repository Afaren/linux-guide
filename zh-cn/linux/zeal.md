# 安装 [Zeal](http://zealdocs.org)

## Linux Mint & Ubuntu 14.04 至 15.04

```bash
$ sudo apt-add-repository ppa:zeal-developers/ppa
$ sudo apt-get update
$ sudo apt-get install zeal
```

## Ubuntu 15.10 及以上
```bash
$ sudo apt-get update
$ sudo apt-get install zeal
```

## Deepin

需要单独编译，详情请查看[官方GitHub](https://github.com/zealdocs/zeal)。
